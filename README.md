## Install

```sh
yarn add github2slack
```

## Usage

```
Usage

   $ github2slack <command> <params>

   $ github2slack sample <param>             # Uses the <PARAM>
   
 Examples

   $ github2slack sample TEST                # Uses the TEST
   $ github2slack sample YOLO                # Uses the YOLO
```
